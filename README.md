# Task-11-Phishing-Attack-Simulation-Detection
**# Objective**
To simulate a phishing attack using GoPhish and analyze how users interact with phishing emails.
-----
**# Tools Used**
GoPhish
Kali Linux
Web Browser
-----
**# Steps Performed**
Created a fake phishing email template.
Set up a phishing landing page.
Launched a phishing campaign.
Tracked email opens, clicks, and credentials.
Identified phishing red flags.
Learned prevention techniques.
-----
**# Outcome**
Improved understanding of phishing attacks and social engineering awareness.
-----
**# Disclaimer**
This simulation was conducted strictly for educational purposes.
